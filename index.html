Hello World!

The purpose of this project was to analyze how weather changes as you get closer to the equator. We pulled data on over 500 cities from the OpenWeatherMap API to create the dataset to accomplish this.


Once the dataset was assembled, we used Matplotlib to plot the relationships between various weather aspects and latitude. The specfiic factors included were: temperature, cloudiness, wind speed, and humidity. This site contains the visualizations and source data from our analysis, as well as explanations and descriptions of any trends we found.
